# Candidate Documentation

V1 Next.js generated from the Mistral Vibe React canvas.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Push to GitHub

```bash
git init
git add .
git commit -m "feat: add candidate documentation homepage"
git branch -M main
git remote add origin https://github.com/tewfiq/doc.git
git push -u origin main
```
